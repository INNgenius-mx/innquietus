# Competitive Landscape Analysis

Comprehensive frameworks for analyzing competition, identifying differentiation opportunities, and developing winning market positioning strategies.

## Use this skill when

- Working on competitive landscape analysis tasks or workflows
- Needing guidance, best practices, or checklists for competitive landscape analysis
- Analyzing competitors and market positioning
- Developing differentiation strategies

## Do not use this skill when

- The task is unrelated to competitive landscape analysis
- You need a different domain or tool outside this scope

## Core Frameworks

### 1. Porter's Five Forces
Analyze industry attractiveness and competitive intensity:

| Force | Key Questions |
|-------|---------------|
| **Rivalry** | How intense is competition? What are competitors doing? |
| **New Entrants** | How easy is it to enter the market? What are barriers? |
| **Substitutes** | What alternatives exist? How easily can customers switch? |
| **Buyer Power** | How much leverage do customers have? |
| **Supplier Power** | How much leverage do suppliers have? |

### 2. Competitive Matrix
Map competitors across key dimensions:

| Dimension | Your Product | Competitor A | Competitor B |
|-----------|--------------|--------------|--------------|
| Price | | | |
| Features | | | |
| Quality | | | |
| Support | | | |
| Market Share | | | |

### 3. SWOT Analysis
For each competitor:
- **Strengths**: What are they good at?
- **Weaknesses**: Where do they fall short?
- **Opportunities**: What market gaps exist?
- **Threats**: What could disrupt them?

### 4. Positioning Map
Create 2x2 matrices to visualize:
- Price vs. Quality
- Features vs. Ease of Use
- Enterprise vs. SMB focus
- Innovation vs. Stability

## Analysis Process

### Phase 1: Identification
1. List direct competitors (same solution)
2. List indirect competitors (same problem, different solution)
3. List potential future competitors
4. Identify market leaders and challengers

### Phase 2: Research
Gather data on each competitor:
- Pricing and packaging
- Feature set and roadmap
- Target audience and positioning
- Marketing channels and messaging
- Reviews and customer sentiment
- Funding and team size

### Phase 3: Differentiation
Identify opportunities:
- Underserved segments
- Feature gaps
- Pricing opportunities
- Positioning angles
- Service/support differentiation

### Phase 4: Strategy
Develop your competitive response:
- Positioning statement
- Key differentiators
- Competitive talk tracks
- Win/loss analysis framework
